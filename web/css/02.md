css 命令整理
###一.CSS概述
	+ 层叠样式表
	+ 不需要编译，直接被浏览器解释执行
	+ 用来实现内容与显示的分离
		+ 改版简单，维护方便
		+ html与css分离开发方便
		+ 减少代码量，增加加载速度
	+ 前端开发必备的技能
###二.CSS引入方式
	+ 在HTML文件的head部分加入
		+ <link rel="stylesheet" type="text/css" href=“my.css"/>
			+ rel 表明链接的文件与html文件之间的关系
			+ type 表明文件类型
			+ href 指向链接的css 外部文件
	+ 在HTML文件的head部分直接写入css文档
		<style type="text/css">
			div {
				margin:0;
				padding:0;
				border:1px solid red;
			}
		</style>		
	+ 在HTML标签里直接写入css属性
		<div style="border:1px solid red; height:30px;"></div>
	+ 使用@import 引入css 文件
		@import url(my.css)
		PS: @import 本身为css 命令 在能在css 文档中使用
